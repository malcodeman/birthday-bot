Birthday Bot is a slack app for keeping track of office birthdays.

## Getting started

```
git clone https://github.com/malcodeman/birthday-bot.git
cd birthday-bot
yarn install && yarn start
```

.env

```
NOTION_TOKEN
NOTION_DATABASE_ID
SLACK_SIGNING_SECRET
SLACK_BOT_TOKEN
SLACK_CHANNEL_ID
```
